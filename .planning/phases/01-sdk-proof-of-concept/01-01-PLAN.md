# Plan 01-01: SDK Query Wrapper

## Objective
Validate the Agent SDK API surface and build the query wrapper that all downstream components depend on.

## Requirements Covered
- SDK-01: query() call with system prompt preset, settings sources, and permission mode
- SDK-02: Structured JSON output extraction from query responses
- SDK-03: Cost data (total_cost_usd) extraction from SDK result messages
- SDK-04: SDK error categorization (network, auth, budget exceeded)
- SDK-05: Document divergences between SPEC.md pseudocode and actual SDK behavior

## Tasks
1. Define SDK types (QueryOptions, QueryResult, SDKError)
2. Implement executeQuery() with correct SDK configuration
3. Handle message stream with raw async iterator
4. Extract cost from SDK messages
5. Categorize errors by type
6. Add inactivity timeout
7. Document all SPEC.md divergences
